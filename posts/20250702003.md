### ✅ 한눈에 보는 요약

- iOS 시스템에 **‘티머니’ 교통카드 코드 탑재** 확인
- 애플페이 연동 통해 **아이폰으로 NFC 교통 결제 가능해질 예정**
- 출시 예상 시기: **2024년 하반기 ~ 2025년 초**
- 서울시·티머니·애플 간 **요금 수수료 협상 마무리 단계**

### 📌 지금까지 아이폰으로 교통카드를 쓰는 방법은?

| 사용 방식 | 가능 여부 | 불편한 점 |
| --- | --- | --- |
| NFC 티머니 태깅 | ❌ 불가능 | 아이폰은 NFC 개방 제한 |
| QR 코드 결제 (모바일 티머니 앱) | ⭕ 가능 | 속도 느림, 자연스럽지 않음 |
| 티머니 스티커 카드 부착 | ⭕ 가능 | 별도 실물 충전 필요, 번거로움 |
| 실물 티머니 카드 사용 | ⭕ 가능 | 아이폰 연동 없음, 수동 관리 |

> iPhone은 지금까지 NFC 기반 티머니 사용이 공식적으로 불가능했으며, 우회 방법에 의존해야 했습니다.
> 

---

### 🚀 티머니-애플페이 도입 후 어떻게 달라질까?

| 항목 | 변화 내용 |
| --- | --- |
| **등록 방식** | 지갑(Wallet) 앱 → 교통카드 → 티머니 선택 가능 |
| **결제 방법** | iPhone 또는 Apple Watch를 **NFC 단말기에 태그만 하면 결제** |
| **속도** | **Face ID 없이 태그만으로 통과 가능 (Express 모드 적용 시)** |
| **충전 및 잔액 관리** | Apple Pay, 신용카드 등으로 실시간 충전 및 잔액 확인 |
| **앱 연동** | 지갑 앱 및 티머니 전용 앱 연동 가능성 |

---

### ⏱️ 언제부터 가능할까?

- 현재 iOS 시스템 내부에서 **‘TransitNetworkIdentifier = tmoney’** 코드 확인
- **기술적 준비는 완료**, 서울시 및 애플과의 **요금 수수료 협의 중**
- 예상 시점: **2024년 하반기~2025년 초**, 아이폰 신모델 출시 시점과 맞물릴 가능성

### 🧩 개발자 문서에서도 확인된 티머니 도입 정황

Apple 공식 개발자 문서에는 다음과 같은 항목이 명시되어 있습니다:

🔗 [developer.apple.com/documentation/passkit/pkpaymentnetwork/tmoney](https://developer.apple.com/documentation/passkit/pkpaymentnetwork/tmoney)

> A payment network for the Tmoney transit card.
> 
> 
> *(티머니 교통카드를 위한 결제 네트워크)*
> 

이 정의는 **티머니가 Apple Pay의 공식 결제 네트워크로 등록되었음을 의미**합니다.

- 개발자는 이제 **애플페이로 티머니 결제 연동 기능을 구현할 수 있게 되었고**,
- 이는 **서비스 오픈 직전 단계로의 진입을 상징하는 신호**입니다.

    ### 🇯🇵 일본 Suica vs 🇰🇷 한국 Tmoney (Apple Pay 교통카드)
    
    | 항목 | 일본 (Suica) | 한국 (Tmoney, 도입 예정) |
    | --- | --- | --- |
    | **도입 시기** | 2016년 (iOS 10) | 2024년 하반기 ~ 2025년 초 |
    | **운영 주체** | JR East (공기업) | 티머니 (서울시 산하) |
    | **Apple Pay 연동** | 정식 지원, Wallet 등록 가능 | iOS 내 코드 등록 완료, 도입 확정적 |
    | **등록 방식** | Wallet 앱에서 Suica 추가 | Wallet 앱에서 Tmoney 추가 예정 |
    | **충전 방식** | Apple Pay / 자동충전 지원 | Apple Pay, 카드 등 충전 연동 예상 |
    | **Express Transit** | ✅ 지원 (잠금 없이 태그 결제) | ✅ 동일 기능 적용 가능성 높음 |
    | **사용 범위** | 지하철, 버스, 자판기, 편의점 | 버스, 지하철 (향후 확장 가능성) |
    | **정기권 기능** | ✅ 가능 (노선 설정/변경) | ❓ 미정 (추후 가능성 있음) |
    | **지갑 앱 통합 UX** | ✅ 잔액, 거래내역, 충전 등 통합 | ✅ 유사한 UX 기대됨 |
    
    ---
    
    ### 💬 간단 요약
    
    - 일본 Suica는 **아이폰 하나로 교통과 결제 모두 처리하는 대표 사례**
    - 한국도 티머니가 Apple Pay에 등록됨에 따라, **Suica와 유사한 경험 제공 가능성 높음**
    - 특히 **Express Transit 모드, 실시간 충전, 지갑 앱 통합 UX**가 핵심 유사점
