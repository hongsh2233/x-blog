말로만 듣던 터미널용 AI 도우미, **Gemini CLI**를 직접 써봤습니다!

미래에셋생명 프로젝트의 문서 정리 작업을 하면서 실제로 적용해봤는데요.

물론, 몇 번은 명령어를 다르게 이해하여 작업을 번복하는 일도 있었지만요 😅

## 💼 실사용 후기: 미래에셋생명 프로젝트 가이드 추출

### ✔️ 어떤 작업에 썼나요?

- 프로젝트 가이드 일부를 PDF에서 추출해서, 정리/요약하는 작업에 활용했습니다.

### 📌 어떻게 사용했는지

- 터미널에서 `gemini` 명령어 입력 → 파일 지정 → 코드 분석 요청
- 예를 들어, `calendar.js`라는 파일에 대해 이런 식으로 요청했어요:
- 명령 예시:
    
    ```bash
    bash
    calendar.js 내용을 요약해서 설명해줘
    ```

## ✅ 좋았던 점

- 설치가 간단해서 바로 써볼 수 있음
- 코드 설명이나 요약이 꽤 정확함
- 기본적인 검색이나 파일 탐색 기능도 있어 편리함
- 반복되는 분석 작업을 자동화해서 효율이 확실히 올라감

## 🤔 흥미로웠던 포인트

질문을 다시 던지면, **이전 답변에 사과하면서** 내용을 바로잡기도 해요. 뭔가 인간적인(?) AI 느낌.

## ⚠️ 아쉬운 점

- 파일 이동이나 스크롤을 할 때마다 **터미널에 내용이 계속 쌓임** → 가독성 떨어짐
    
    ![image.png](/images/posts/20250715002/08.png)
    
- **모든 코드가 Google로 전송**된다는 점이 조금 민감할 수 있음 (Google로 전송되어 AI 학습에 이용됨)
    
    → `.gitignore`로 예외 처리하거나 유료 API Key로 ‘학습 제외’ 옵션 설정 필요
    
    → 보안 민감한 환경에서는 주의 필요!
