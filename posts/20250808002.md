## 🚀 Figma Make란?

Figma Make는 **자연어 프롬프트**와 기존 디자인 정보를 기반으로

UI 컴포넌트, 페이지, 프로토타입까지 자동 생성해 주는 AI 기능입니다.

> 디자인 경험이 없어도, 빠르게 시안을 구현할 때 유용합니다.
>

## 결과

- 스타일 속성 **거의 완벽 일치**
- 구조(타이틀·텍스트·버튼) **정확 재현**
- 브라우저 렌더링 차이로 미세 간격 차이 발생
- 오타(`color:#fff;j`)까지 그대로 반영됨 → 조건 입력 전 검수 필요

---

## 📦 Make가 보여준 React 코드

### 🗂 코드 구조 요약

- **`App.tsx`**: 팝업 열림/닫힘 상태(`isPopupOpen`) 관리, 버튼 클릭 시 팝업 표시
- **`LayerPopup.tsx`**: 팝업 UI 컴포넌트. 열림 여부, 닫기 이벤트, 타이틀·본문·버튼 텍스트를 `props`로 받아 렌더링

---

### 🎨 스타일

프롬프트에 입력한 CSS 조건을 **인라인 스타일**로 그대로 반영

- 오버레이(dim): `#00000080`
- 팝업 박스: `border-radius: 16px; padding: 48px 32px 24px; background: #fff;`
- 타이틀·본문·버튼: 지정한 색상, 폰트, 크기, 라인 높이 모두 적용
- 버튼은 `border-radius: 100px`으로 생성

---

### ⚙️ 동작

1. “팝업 열기” 버튼 클릭 → 팝업 오픈
2. 버튼 클릭 시 `onButtonClick` 실행
3. 닫기 버튼 클릭 또는 `onClose()` 호출 시 닫힘

---

### 🔍 특이점

- **글로벌 변수 활용**
    
    `:root`에 색상·폰트·간격 등 공통 스타일을 **CSS 변수**로 선언 → 전체 UI에서 재사용 가능
    
- **조건별 인라인 스타일**
    
    프롬프트 입력값을 코드에 즉시 반영, 수정 속도 ↑

### App.tsx

```tsx
/*App.tsx*/
import React, { useState } from 'react';
import { LayerPopup } from './components/LayerPopup';

export default function App() {
  const [isPopupOpen, setIsPopupOpen] = useState(false);

  const openPopup = () => {
    setIsPopupOpen(true);
  };

  const closePopup = () => {
    setIsPopupOpen(false);
  };

  const handleButtonClick = () => {
    alert('팝업 버튼이 클릭되었습니다!');
  };

  return (
    <div className="size-full flex items-center justify-center bg-gray-100">
      <div className="text-center">
        <h1 className="mb-8">레이어 팝업 데모</h1>
        
        <button
          onClick={openPopup}
          className="px-6 py-3 bg-blue-500 text-white rounded-lg hover:bg-blue-600 transition-colors"
        >
          팝업 열기
        </button>
      </div>

      <LayerPopup
        isOpen={isPopupOpen}
        onClose={closePopup}
        title="팝업 타이틀"
        content="이것은 팝업의 설명 텍스트입니다. 사용자에게 중요한 정보를 전달하거나 확인을 요청할 때 사용됩니다."
        buttonText="확인"
        onButtonClick={handleButtonClick}
      />
    </div>
  );
}
```

    export function ImageWithFallback(props: React.ImgHTMLAttributes<HTMLImageElement>) {
      const [didError, setDidError] = useState(false)
    
      const handleError = () => {
        setDidError(true)
      }
    
      const { src, alt, style, className, ...rest } = props
    
      return didError ? (
        <div
          className={`inline-block bg-gray-100 text-center align-middle ${className ?? ''}`}
          style={style}

          </div>
        </div>
      ) : (
        <img src={src} alt={alt} className={className} style={style} {...rest} onError={handleError} />
      )
    }
