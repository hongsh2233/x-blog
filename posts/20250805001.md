## 🧠 독자 AI 파운데이션 모델 프로젝트란?

> “처음부터 끝까지, 우리 손으로 만드는 한국형 AI 모델!”
> 

과학기술정보통신부가 주관하는 프로젝트로, ChatGPT 같은 대형 언어모델(LLM)을 **한국이 처음부터 끝까지 직접 개발**하기 위한 사업입니다.

해외 모델을 가져와 조금 수정하는 것이 아니라,
” 모델 설계부터 데이터 수집, 학습까지 ‘완전한 국산화’ ”가 목표입니다.

[1] fine-tuning 방식 : 이미 만들어진 AI 모델에 새로운 데이터를 학습시켜, 특정 용도에 맞게 조정하는 과정

## 📅 「독자 AI 파운데이션 모델」 프로젝트 추진 일정

| 날짜 | 내용 |
| --- | --- |
| **2025.06.20** | 사업 공고 시작 (공식 사업계획 공고) |
| **2025.06.28** | 사업설명회 개최 (참여기업 대상 설명회) |
| **2025.07.04** | 제안서 접수 마감 |
| **2025.07.08 ~ 07.12** | 서면평가 실시 (15개 → 10개 정예팀 압축) |
| **2025.07.25** | 서면평가 결과 발표 (10개 정예팀 선정) |
| **2025.07.26 ~ 08.01** | 발표평가 진행 (10개 팀 대상) |
| **2025.08.05** | 최종 5개 정예팀 선정 발표 |
| **2025.08 ~ 12월** | 사전 모델 설계, 데이터 검증 및 개발 준비 |
| **2026년 상반기~** | 사전학습 모델 공개 및 실증 적용 단계 예정 |

### 🧾 참고 사항

- **평가 절차**는 '서면평가 → 발표평가 → 최종선정'의 3단계 구조로 진행되었습니다.
- 최종 선정된 5개 팀은 이후 **사전 모델 설계 및 공동 검증 체계에 참여**하게 되며,
    
    실제 **모델 학습과 실증 적용은 2026년부터 본격화**됩니다.

> 🏅 이들은 **'국가대표 AI' 모델 개발을 주도할 핵심 5개 팀** 으로 선정되었으며, 
향후 공동 모델 개발, 검증, 실증 사업을 함께 수행합니다.

## ⚠️ 향후 계획 : 최종 2개 팀 선정?

- 일부 언론에서는 **5개 팀 중 2개 팀이 최종 개발 주체로 선정될 수 있다**는 가능성을 보도했습니다.
- **하지만 과학기술정보통신부의 공식 발표 자료(2025.08.05 기준)에는
“2개 팀으로 압축” 혹은 “공동 개발 주체 선정” 등의 표현은 포함되어 있지 않습니다.**

> ✅ 따라서 현시점에서의 공식적인 최종 선정 결과는 5개 팀 까지 입니다.
>

## 💬 왜 중요한가요?

- ✅ **AI 기술 주권 확보**
    
    외산 모델 의존 없이, 자체 기술로 개발

- ✅ **국내 생태계 강화**
    
    기업, 연구기관, 스타트업까지 함께 성장하는 구조 마련
