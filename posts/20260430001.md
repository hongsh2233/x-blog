## 🚀 기술 블로그

### GSAP 3.15 - easeReverse
이 영상은 GSAP 3.15 버전에서 새롭게 도입된 easeReverse 기능을 소개하며, 더욱 자연스럽고 사용자 친화적인 웹 UI 인터랙션을 만드는 방법을 설명합니다.

- 주요 내용 요약:
    
     easeReverse의 필요성 (0:05 - 4:21): 기존에는 애니메이션을 reverse()로 되돌릴 때 이징(easing)도 함께 반전되어 부자연스러운 움직임(예: ease-out이 ease-in으로 변함)이 발생했습니다. 이를 해결하기 위해 사용자는 이징을 수동으로 교체하거나 새 애니메이션을 생성해야 했는데, 이는 코드 복잡도를 높이고 진행 상황 인지 문제를 야기했습니다.
    
     easeReverse란? (4:22 - 5:16): 이 기능을 사용하면 애니메이션의 재생 방향이 바뀔 때 이징을 독립적으로 재설정할 수 있습니다. 덕분에 '빠른 진입, 부드러운 퇴장' 같은 최적의 UX를 구현할 수 있습니다.    

     다양한 애니메이션 패턴 (9:32 - 16:12):
    

     제어 메서드 활용: play()와 reverse()를 사용해 간단한 호버나 드롭다운 인터랙션을 만드는 방식입니다.

     동적 트윈(Dynamic Tweening): 자석 버튼처럼 매번 타겟 값이 달라지는 경우에 적합합니다.
    
     타임라인 클리어 및 재구축: 완전히 다른 입구와 출구 시퀀스가 필요할 때 사용합니다.
    
     하이브리드 패턴: easeReverse와 타임라인의 addPause를 조합하여, 사용자가 애니메이션 도중 방향을 바꾸면 자연스럽게 되돌아가고, 끝까지 실행되면 완전히 다른 종료 시퀀스가 작동하도록 구성할 수 있습니다.
    

     핵심 UX 원칙:
    
     즉각적인 중단 가능성: 사용자의 행동을 차단하지 않도록 애니메이션은 언제든 취소하거나 방향을 전환할 수 있어야 합니다 (2:26).
    
     Fancy In, Quick Out: 화려한 진입 애니메이션 후에는 사용자가 기다리지 않도록 빠른 퇴장 애니메이션을 적용하는 것이 좋습니다 (2:41).

### 최신 CSS 기능 3가지
웹사이트 구축 시 브라우저 간의 불일치를 줄이고, 더 나은 사용자 경험(UX)을 위해 리셋 CSS에 추가해야 할 필수 속성들입니다.

- scrollbar-gutter: stable — 레이아웃 점프 방지
    - 콘텐츠가 길어져 스크롤바가 생길 때, 화면이 왼쪽으로 미세하게 밀리는 현상을 해결합니다.

    - 효과: 스크롤바가 없더라도 미리 그만큼의 공간(gutter)을 확보하여, 페이지 이동이나 팝업 시 레이아웃이 덜컥거리는 현상을 원천 차단합니다.

    - 설정: html { scrollbar-gutter: stable; }

- min-block-size: 100svb — 완벽한 모바일 뷰포트 대응  
모바일 브라우저의 상단 바/하단 바 유무에 따라 높이가 달라지는 문제를 해결하는 가장 최신 단위입니다.

    - 효과: 100vh 사용 시 모바일 바에 가려지던 하단 영역 문제를 해결합니다.  
    svb(Small Viewport Block-size) 단위를 사용하면 논리적 방향(Block)에 맞춰 현재 가시적인 가장 작은 뷰포트 크기를 보장합니다.

    - 설정: body { min-block-size: 100svb; }

- interpolate-size: allow-keywords — auto 높이 애니메이션의 해방  
CSS의 오랜 숙제였던 height: auto로의 전환 애니메이션을 가능하게 합니다.

    - 효과: 과거에는 height: 0에서 auto로 변할 때 transition이 작동하지 않았으나, 이 속성을 쓰면 max-content나 auto 같은 키워드 값 사이의 부드러운 애니메이션을 브라우저가 직접 계산합니다.

    - 설정: :root { interpolate-size: allow-keywords; }

- 코드 변환 지원: 디자인 결과물을 React, Tailwind CSS 등 실제 개발 코드로 즉시 내보내기 가능
