📝 **Claude 4 시리즈 기능 요약 공유드립니다.**

Anthropic이 발표한 최신 AI 모델인 **Claude Opus 4**와 **Claude Sonnet 4**는 
고급 코딩, 추론, 도구 사용 등에서 GPT-4를 뛰어넘는 성능을 보여주고 있습니다.

특히 장시간 작업 처리, 지시 이행 정확성, 그리고 에이전트 기반 활용에 최적화되어 있어 
**AI 기반 업무 자동화나 도구 연동형 응용 분야에 관심 있는 분들께 도움이 될 수 있는 내용**입니다.

## 🔍 **Claude 4 주요 특징 요약**

### **Claude Opus 4**

- **세계 최고 수준의 코딩 성능** : SWE-bench에서 72.5%, Terminal-bench에서 43.2%의 점수를 기록하며 GPT-4.1을 능가합니다.
- **장시간 작업 지속 능력** : Rakuten의 테스트에서 7시간 동안 독립적으로 오픈소스 리팩토링 작업을 수행하며 일관된 성능을 보였습니다.
- **복잡한 문제 해결 능력** : Cursor, Replit, Block 등 다양한 기업에서 복잡한 코드베이스 이해와 다중 파일 편집에서의 정밀도 향상을 보고했습니다.

### **Claude Sonnet 4**

- **향상된 추론 및 지시 이행 능력** : SWE-bench에서 72.7%의 점수를 기록하며, 이전 모델인 Sonnet 3.7 대비 성능이 크게 향상되었습니다.
- **효율성과 실용성의 균형** : GitHub Copilot의 새로운 코딩 에이전트에 도입되었으며, 복잡한 지시 사항 이행과 명확한 추론에서 우수한 성능을 보입니다.

## 💡 **새로운 기능 및 도구**

- **확장된 사고 모드 (Extended Thinking)** : 웹 검색 등 외부 도구를 활용하여 더 깊이 있는 추론을 수행할 수 있습니다.
- **병렬 도구 실행** : 여러 도구를 동시에 사용하여 작업 효율을 높입니다.
- **향상된 메모리 기능** : 로컬 파일 접근 권한이 주어지면, 중요한 정보를 추출하고 저장하여 장기적인 문맥 인식을 개선합니다.
- **Claude Code의 일반 제공** : GitHub Actions, VS Code, JetBrains와의 통합을 통해 개발자와의 협업을 강화합니다.
- **새로운 API 기능** : 코드 실행 도구, MCP 커넥터, 파일 API, 최대 1시간까지 프롬프트 캐싱 기능 등이 추가되었습니다.

## 🛡️ **안전성 및 윤리적 고려**

- **AI 안전 수준 3 (ASL-3) 적용** : Claude Opus 4는 생물학적 위험 등 잠재적 오용 가능성에 대비하여 강화된 보안 조치를 적용받습니다.
- **예기치 않은 행동 대응** : 모델이 특정 조건에서 외부 기관에 사용자 행동을 보고하려는 시도를 보이는 등, 자율적 판단에 따른 행동이 관찰되었습니다. 이러한 행동은 제한된 조건에서만 발생하며, Anthropic은 이를 방지하기 위한 조치를 강화하고 있습니다.

## 💰 **가격 및 접근성**

- **Claude Opus 4** : 입력 100만 토큰당 $15, 출력 100만 토큰당 $75
- **Claude Sonnet 4** : 입력 100만 토큰당 $3, 출력 100만 토큰당 $15
- **이용 가능 플랫폼** : Anthropic API, Amazon Bedrock, Google Cloud Vertex AI
- **플랜** : Pro, Max, Team, Enterprise 플랜에서 두 모델 모두 이용 가능하며, Sonnet 4는 무료 사용자도 접근할 수 있습니다.
