## Tailwind CSS란?

**Tailwind CSS**는 유틸리티 퍼스트(Utility-First) CSS 프레임워크입니다. 미리 정의된 클래스를 조합하여 빠르게 UI를 구성할 수 있고, 프로덕션 빌드 시 사용하지 않는 CSS를 자동으로 제거(purging)하여 번들 크기를 최소화합니다.

## 반응형 디자인

Tailwind의 반응형 시스템은 모바일 퍼스트(Mobile First) 방식입니다. 접두사 없이 작성한 스타일이 기본(모바일)이고, 브레이크포인트 접두사로 더 큰 화면을 대응합니다.

```html
<!-- 모바일: 1열, 태블릿(md): 2열, 데스크탑(lg): 3열 -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
  <div class="p-4 bg-white rounded-lg shadow">카드 1</div>
  <div class="p-4 bg-white rounded-lg shadow">카드 2</div>
  <div class="p-4 bg-white rounded-lg shadow">카드 3</div>
</div>

<!-- 텍스트 크기 반응형 적용 -->
<h1 class="text-2xl md:text-4xl lg:text-5xl font-bold">
  반응형 제목
</h1>
```

## 커스텀 테마 설정

`tailwind.config.ts`에서 디자인 토큰을 정의하여 프로젝트 전반에 일관된 스타일을 적용할 수 있습니다.

```typescript
// tailwind.config.ts
import type { Config } from 'tailwindcss';

const config: Config = {
  theme: {
    extend: {
      colors: {
        brand: {
          50:  '#eff6ff',
          500: '#3b82f6',
          900: '#1e3a8a',
        },
      },
      fontFamily: {
        sans: ['Pretendard', 'sans-serif'],
        mono: ['JetBrains Mono', 'monospace'],
      },
      spacing: {
        '18': '4.5rem',
        '88': '22rem',
      },
    },
  },
};

export default config;
```

## 다크모드

Tailwind는 `dark:` 변형자를 통해 다크모드를 간단하게 구현할 수 있습니다.

```html
<!-- 배경/텍스트 자동 전환 -->
<div class="bg-white dark:bg-gray-900 text-gray-900 dark:text-gray-100">
  <h1 class="text-gray-800 dark:text-white">다크모드 제목</h1>
  <p class="text-gray-600 dark:text-gray-400">설명 텍스트</p>
</div>
```

```typescript
// tailwind.config.ts - 클래스 기반 다크모드 설정
const config: Config = {
  darkMode: 'class', // 'media' 대신 'class'로 수동 제어
};
```

## `@apply`로 컴포넌트 추출

자주 반복되는 클래스 조합은 CSS에서 `@apply`로 추출할 수 있습니다.

```css
/* globals.css */
@layer components {
  .btn-primary {
    @apply px-4 py-2 bg-blue-600 text-white rounded-lg
           hover:bg-blue-700 transition-colors duration-200
           focus:outline-none focus:ring-2 focus:ring-blue-500;
  }

  .card {
    @apply bg-white dark:bg-gray-800 rounded-xl shadow-sm
           border border-gray-200 dark:border-gray-700 p-6;
  }
}
```

## 정리

Tailwind CSS는 처음엔 클래스가 많아 낯설지만, 익숙해지면 CSS 파일을 별도로 관리하지 않아도 되어 생산성이 크게 향상됩니다. 커스텀 테마와 컴포넌트 추출을 적절히 활용하여 유지보수하기 좋은 코드를 작성하세요.
